# Streamlit RAG Chatbot App

This is a Retrieval-Augmented Generation (RAG) chatbot using Streamlit, LangChain, and ChromaDB.

## Features
- Local embeddings via sentence-transformers
- Chroma vector store
- Streamlit web UI
